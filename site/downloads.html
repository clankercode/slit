build from source

Compile it yourself.

C
$ git clone https://github.com/clankercode/slit
$ cd slit/c
$ make
$ ./slit --version
slit 0.1.0 (c)
Rust
$ git clone https://github.com/clankercode/slit
$ cd slit/rust
$ cargo build --release
$ ./target/release/slit --version
slit 0.1.0 (rust)
Go
$ git clone https://github.com/clankercode/slit
$ cd slit/go
$ go build -o slit ./...
$ ./slit --version
slit 0.1.0 (go)
