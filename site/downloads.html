build from source

Compile it yourself.

C
$ git clone https://github.com/clankercode/slit
$ cd slit/c
$ make
$ ./slit --version
slit {{VERSION}} (c)
Rust
$ git clone https://github.com/clankercode/slit
$ cd slit/rust
$ cargo build --release
$ ./target/release/slit --version
slit {{VERSION}} (rust)
Go
$ git clone https://github.com/clankercode/slit
$ cd slit/go
$ go build -o slit ./...
$ ./slit --version
slit {{VERSION}} (go)
